These are some basic training terms that you should be familiar with:
1. Workout: the training session that contains an X number of exercises.
2. Set: a set of X number of repetitions for a given exercise. For example, 3 sets of 10 pushups means that each 10 pushups is a completion of one set.
3. Repetition (rep): for easy definition, a repetition is a one complete movement of a given exercise. In the previous example, 3 sets of 10 pushups means we do 10 reps of pushups in each set.
4. Rest: How much you rest between sets or between exercises.
5. Round: This means doing 1 set of each diffenet exercise and then repeating the round for X number of rounds. You can rest a little bit between each exercise or between rounds.

Training is a lifetime journey. There are smart training methods, but there are no shortcuts for success. Committing to your workouts and not skipping any one of them is a key principle in achieving success in the long run.
